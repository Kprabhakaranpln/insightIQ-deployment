# 🧠 InsightIQ | Data Refinery

InsightIQ is a web-based data engineering and analytics tool built with Python and Streamlit. It allows users to upload raw datasets and instantly clean, process, and visualize the information without writing a single line of code.

## 🚀 Live Demo
[Click here to view the live app!](https://insightiq-deployment-am4z2z2bvitnbmpxgydm56.streamlit.app)

## 🛠️ Features
* **Multi-Format Support:** Ingest CSV, Excel, JSON, Parquet, and Avro files.
* **Automated Cleaning:** Remove duplicates and handle missing data with one click.
* **Interactive Visualizations:** Generate dynamic charts and dashboards using Plotly.
* **Secure Export:** Download processed datasets and human-in-the-loop analyst notes.

## 💻 Tech Stack
* **Language:** Python
* **Framework:** Streamlit
* **Data Processing:** Pandas
* **Visualization:** Plotly Express
